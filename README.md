# 网页自动刷新

## 核心功能

1. **定时自动刷新**：支持设置网页自动刷新时间，满足实时数据监控、抢单等场景需求
2. **多单位时间设置**：提供 "秒 (s)、分 (m)、时 (h)" 三种时间单位选择，灵活适配不同刷新频率需求
3. **状态记忆功能**：刷新页面后自动恢复之前的运行状态、时间设置和面板位置，无需重复配置

## 交互体验

1. **自由拖拽面板**：通过标题栏可将工具面板拖动到页面任意位置，避免遮挡关键内容
2. **实时倒计时显示**：清晰展示距离下次刷新的剩余时间，格式随时长智能变化（如 "30s"、"2m15s"、"1h5m"）
3. **状态切换反馈**：通过按钮颜色（蓝→红）、图标（▶→⏸）和文字变化，直观展示当前运行状态

## 视觉设计

1. **精致 UI 设计**：采用渐变色彩、细腻阴影和圆角设计，符合现代审美
2. **丝滑过渡动画**：下拉框箭头旋转、按钮悬停缩放、面板拖拽反馈等操作均有流畅动画
3. **响应式状态变化**：元素交互时（聚焦、悬停、点击）有明确的视觉反馈，提升操作体验

## 数据管理

1. **按网站独立配置**：不同网站自动保存各自的刷新设置，互不干扰
2. **最小值限制**：防止误设置过短时间（秒≥5、分≥1、时≥1），避免过度刷新
3. **本地持久化**：所有设置保存在本地，无需担心数据丢失

## 使用场景

- 实时数据监控（股票、赛事、统计数据等）
- 抢单、抢购等需要及时刷新页面的场景
- 监控网页内容更新（公告、通知等）
- 自动化测试辅助工具
